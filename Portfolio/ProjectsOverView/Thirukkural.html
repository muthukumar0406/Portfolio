Thirukkural Explore

Technologies Used

  • ASP.NET Core Web APi
  • SQL Server
  • JavaScript
  • Bootstrap
  • HTML&&CSS

Key Features:

  • GET: Fetch all Thirukkurals.
  • GET by ID: Retrieve Kural by its number.
  • POST: Add a new Kural (PostRange API support).
  • DELETE: Remove a specific Kural by ID.

Responsibilities:

  • Built and deployed RESTful APIs using ASP.NET Core Web API.
  • Implemented routing and controllers to handle Kural data.
  • Developed a frontend using JavaScript to fetch and display data.
  • Designed the user interface using HTML, CSS, and Bootstrap.
  • Ensured smooth integration between frontend and backend.
